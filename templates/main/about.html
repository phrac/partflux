About Partfindr / F.A.Q.

General information

1.1 What is Partfindr?
Partfindr is a service that helps people find information about any part number. This information includes cross references, attributes, and places where the part can be purchased.


1.2 What is considered a part?
Just about anything can be considered a part. If it has a part number, we want to know about it, no matter how trivial it may be. Do you know the part number for that tiny screw that holds the screen of your smart phone together? If so, we want to know about it, and someone else may want to know about it too.


1.2.1 What is an attribute?
An attribute could be something like a shape, color, weight, or height of a part. An attribute is simply a piece of information that describes the part in more detail. You can add attributes to any part.


1.2.2 What is a cross reference?
A cross refernce is a link to another part with similar characteristics that can be used in place of the original part. Think of an oil filter on a car. There are several brands of oil filter that will fit one particular car. All those different oil filters would be cross references of each other. They may be slightly different in color or shape or filtration capacity, but they will all fit in that particular car.


1.3 Where does part data come from?
Information about parts comes from several sources. One method is web robots that continually add new parts to the database, but our preferred method is part data that is entered by our users. Our users are our most valued source of information and we encourage everyone to help make partfindr better by adding new parts to our database, contributing part images, and verifying cross reference information.


1.4 How is data verified?
Our data is verified by the community. We try to ensure that data is 100% accurate and we rely on our users to make this possible. Please contribute and validate information about parts if you are able. User contributions are what make this site possible.


Technical Information

2.1 Who developed this site?
This site is developed and maintained by disflux, LLC owned by Derek and Rebecca Musselmann.


2.2 Why was it developed?
The idea came to me (Derek) while working at my job as a purchasing agent for a local manufacturing company in Omaha, Nebraska. I was often tasked with finding materials for various pieces of manufacturing equipment, with no easily accessible source of data to locate them. This gave me the idea to start a web site dedicated to quick and easy part information retrieval.


2.3 What technology does partfindr use?
Partfindr is programmed in Python using the Django framework. We use several Django packages to provide additional functionality. Data is stored in a PostgreSQL database and search is provided by ElasticSearch. Page serving is handled by nginx. All this running in a private VPS from ARP Networks running FreeBSD 9. Media hosting and storage is handled by Amazon Simple Storage Service (S3).

For web data extraction we currently use Scrapy, a Python framework for web scraping.


2.4 Do you have an API?
Currently, the API is under development. We plan to release this for developers in early 2013.


2.5 Is this site open source?
Open source software is a major component of Partfindr and we plan to release the source for this site at some point, but as of right now it is not open sourced.
